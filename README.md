# Student Performance Analyzer

A beginner-friendly Python project that analyzes students' marks using Pandas and visualizes performance trends using Matplotlib.

This project helped me understand basic data analysis, data cleaning, and plotting techniques.

---

## Features
- Reads student marks from a CSV file  
- Calculates average, minimum, and maximum scores  
- Generates bar charts and performance graphs  
- Provides simple performance insights  

---

## Technologies Used
- Python  
- Pandas  
- Matplotlib
